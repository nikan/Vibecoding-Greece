# Vibecoding-Greece 🇬🇷

Καλώς ήρθατε στο αποθετήριο [**Vibecoding-Greece**](https://www.facebook.com/groups/923120496824057)! Αυτό είναι το επίσημο δημόσιο αποθετήριο για την ομάδα Facebook Vibecoding-Greece, αφιερωμένο στη φιλοξενία και την προβολή έργων που δημιουργούνται από τα μέλη της κοινότητάς μας.

## 📖 Σχετικά

Αυτό το αποθετήριο λειτουργεί ως κεντρικός κόμβος για τη συλλογή και την ανταλλαγή συνδέσμων έργων από μέλη της ομάδας Facebook Vibecoding-Greece. Είτε εργάζεστε σε ανάπτυξη ιστοσελίδων, εφαρμογές για κινητά, επιστήμη δεδομένων ή οποιοδήποτε άλλο έργο προγραμματισμού, αυτό είναι το μέρος για να μοιραστείτε τη δουλειά σας με την κοινότητα!

## 🎯 Σκοπός

- **Προβολή Έργων**: Εμφάνιση της εκπληκτικής δουλειάς που γίνεται από τα μέλη της κοινότητάς μας
- **Συνεργασία**: Ανακαλύψτε έργα και πιθανές ευκαιρίες συνεργασίας
- **Μάθηση**: Μάθετε από τα έργα και τις υλοποιήσεις των άλλων
- **Δικτύωση**: Συνδεθείτε με συναδέλφους προγραμματιστές στην Ελλάδα

## 🚀 Πώς να Συνεισφέρετε

Καλωσορίζουμε συνεισφορές από όλα τα μέλη της ομάδας Facebook Vibecoding-Greece!

### Νέος Τρόπος Υποβολής (GitHub Issues)

[![Submit Project](https://img.shields.io/badge/-Submit_Project-2ea44f?style=flat&logo=github)](https://github.com/nikan/Vibecoding-Greece/issues/new?template=project)

1. **Κάντε κλικ στο παραπάνω κουμπί** - Ανοίγει απευθείας τη φόρμα υποβολής
2. **Συμπληρώστε τα στοιχεία** - Όνομα έργου, URL, περιγραφή, τεχνολογίες, κατηγορία
3. **Υποβάλετε** - Το issue θα εξεταστεί και θα προστεθεί αυτόματα στη λίστα
4. **Για ενημερώσεις** - Επιλέξτε το checkbox "This is an update" στη φόρμα

### Παλιός Τρόπος (Pull Request)

Αν προτιμάτε τον κλασικό τρόπο:
1. **Κάντε Fork** αυτό το αποθετήριο
2. **Προσθέστε το έργο σας** στην κατάλληλη ενότητα
3. **Δημιουργήστε Pull Request** με σαφή περιγραφή

## 📂 Κατηγορίες Έργων

Τα έργα θα οργανώνονται στις ακόλουθες κατηγορίες:

- **Web development**: Ιστοσελίδες, εφαρμογές ιστού και εργαλεία ιστού
- **Mobile development**: Εφαρμογές iOS, Android και cross-platform
- **Backend/APIs**: Εφαρμογές server-side και REST/GraphQL APIs
- **Data Science/ML**: Έργα μηχανικής μάθησης, ανάλυσης δεδομένων και οπτικοποίησης
- **DevOps/Tools**: Εργαλεία ανάπτυξης, scripts αυτοματισμού και υποδομή
- **Άλλα**: Έργα που δεν εμπίπτουν στις παραπάνω κατηγορίες

### [DomainsReseller MCP Server](https://github.com/Webhosting4U/Domain-Names-MCP-Server)
**Συγγραφέας**: Webhosting4U  
**Περιγραφή**: MCP server για διαχείριση domain names από το IDE με φυσική γλώσσα. 32 εργαλεία για καταχώρηση, μεταφορά, ανανέωση, DNS, nameservers και άλλα μέσω του Webhosting4U DomainsReseller API.  
**Τεχνολογίες**: TypeScript, Cloudflare Workers, MCP (Model Context Protocol), Streamable HTTP, Zod  
**Κατάσταση**: Ενεργό
### [AI-Driven Development Lifecycle Framework](https://github.com/toons01/AI-Driven-Dev-Framework)
**Συγγραφέας**: toons01  
**Περιγραφή**: Open-source framework που δίνει δομή στην ανάπτυξη λογισμικού με AI agents. Περιλαμβάνει 22 specialized skills, 7 stack presets, persistent memory bank, quality gates και automated code review — όλα enforced πριν φύγει κώδικας σε production.  
**Τεχνολογίες**: Markdown, Husky, GitHub Actions, ESLint, Playwright, Vitest  
**Κατάσταση**: Ενεργό

### [tsmcp-LSP](https://github.com/nikan/tsmcp-LSP)

**Συγγραφέας**: nikan  
**Περιγραφή**: MCP server που εκθέτει TypeScript semantic analysis (definitions, references, hover, symbols, call hierarchy) ως MCP tools για AI agents, μέσω typescript-language-server.  
**Τεχνολογίες**: TypeScript, Node.js, MCP SDK, LSP, typescript-language-server  
**Κατάσταση**: Ενεργό (v0.1.0)
