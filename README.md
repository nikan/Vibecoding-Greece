## üöÄ How to Contribute

We welcome contributions from all members of the Vibecoding-Greece Facebook group! Here's how you can add your project:

### Adding Your Project

1. **Fork this repository**
2. **Add your project** to the appropriate section (see structure below)
3. **Create a Pull Request** with a clear description of your project
4. **Wait for review** - maintainers will review and merge your submission

### Project Submission Format

When adding your project, please include:

```markdown
### [Project Name](link-to-project)
**Author**: Your Name / GitHub Username  
**Description**: Brief description of what your project does  
**Tech Stack**: Technologies used (e.g., React, Python, Django, etc.)  
**Status**: Active/Completed/In Progress
```

## üìÇ Project Categories

Projects will be organized into the following categories:

- **Web Development**: Websites, web applications, and web tools
- **Mobile Development**: iOS, Android, and cross-platform apps
- **Backend/APIs**: Server-side applications and REST/GraphQL APIs
- **Data Science/ML**: Machine learning, data analysis, and visualization projects
- **DevOps/Tools**: Development tools, automation scripts, and infrastructure
- **Other**: Projects that don't fit into the above categories
